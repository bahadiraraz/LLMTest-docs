# Installation

Install llmtest with the provider you need:

```bash
pip install assertllm[openai]       # OpenAI (GPT-5.x, GPT-4.1, o3, o4-mini)
pip install assertllm[anthropic]    # Anthropic (Claude Opus 4.6, Sonnet 4.6, Haiku 4.5)
pip install assertllm[ollama]       # Local (Ollama — no API key)
pip install assertllm[pydantic-ai]  # Pydantic AI agents
pip install assertllm[all]          # Everything
```

## Package managers

### uv

```bash
uv add assertllm[openai]
```

### PDM

```bash
pdm add assertllm[openai]
```

### Poetry

```bash
poetry add assertllm[openai]
```

### Rye

```bash
rye add assertllm --features openai
```

### Conda + pip

```bash
conda create -n llmtest python=3.12
conda activate llmtest
pip install assertllm[openai]
```

## Requirements

- Python 3.10+
- Pydantic 2.0+

## API keys

```bash
# OpenAI
export OPENAI_API_KEY=sk-...

# Anthropic
export ANTHROPIC_API_KEY=sk-ant-...

# Ollama — no key needed, just run the server
ollama serve
```
