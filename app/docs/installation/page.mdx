# Installation

Install assertllm with the provider you need:

```bash
pip install assertllm[openai]       # OpenAI (GPT-5.x, GPT-4.1, o3, o4-mini)
pip install assertllm[anthropic]    # Anthropic (Claude Opus 4.6, Sonnet 4.6, Haiku 4.5)
pip install assertllm[ollama]       # Local (Ollama — no API key)
pip install assertllm[pydantic-ai]  # Pydantic AI agents
pip install assertllm[all]          # Everything
```

## Requirements

- Python 3.10+
- Pydantic 2.0+
