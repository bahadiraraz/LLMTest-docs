# Getting Started

assertllm is a pytest-based testing framework for LLM applications.
It provides 22+ built-in assertions that are **deterministic and instant** — no paying an LLM to check if your output contains a word.

<Callout type="info">
  **Zero LLM calls for most assertions** — substring, regex, JSON, cost, latency checks are all local and instant.
</Callout>
