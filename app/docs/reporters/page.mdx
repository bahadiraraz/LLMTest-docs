## JUnit Reporter

For GitHub Actions, Jenkins, and other CI systems:

```python filename="report_junit.py"
from llmtest_core.reporters import JUnitReporter

reporter = JUnitReporter()
with open("results.xml", "w") as f:
    f.write(reporter.render(suite_result))
```
