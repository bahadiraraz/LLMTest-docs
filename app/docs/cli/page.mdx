# CLI

## Commands

### `llmtest init`

Create a starter test file:

```bash
llmtest init
```

### `llmtest run`

Run LLM tests:

```bash
llmtest run tests/
llmtest run tests/test_agent.py
```

### `llmtest providers`

List available providers:

```bash
llmtest providers
```
