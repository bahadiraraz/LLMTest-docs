import { Steps, Callout } from 'nextra/components'

# CLI

<Steps>

### Initialize a project

Create a starter test file:

```bash filename="Terminal"
llmtest init
```

### Run LLM tests

```bash filename="Terminal"
llmtest run tests/
llmtest run tests/test_agent.py
```

### Check available providers

```bash filename="Terminal"
llmtest providers
```

```ansi filename="Output"
Provider      Status    Models
──────────────────────────────────────
openai        ✓ ready   gpt-5-mini, gpt-5, gpt-4.1
anthropic     ✓ ready   claude-opus-4-6, claude-sonnet-4-6
ollama        ✗ not running
```

</Steps>

<Callout type="info">
  `llmtest providers` auto-detects which providers are available based on your environment variables and running services.
</Callout>
