# Providers

Install only what you need:

<Cards num={3}>
  <Cards.Card title="OpenAI" href="/docs/providers/openai" arrow />
  <Cards.Card title="Anthropic" href="/docs/providers/anthropic" arrow />
  <Cards.Card title="Ollama (Local)" href="/docs/providers/ollama" arrow />
</Cards>

| Provider | Install | Models |
|---|---|---|
| OpenAI | `pip install "assertllm[openai]"` | GPT-4o, GPT-4o-mini, o3, o4-mini |
| Anthropic | `pip install "assertllm[anthropic]"` | Claude Sonnet 4, Claude Haiku 4.5 |
| Ollama | `pip install "assertllm[ollama]"` | Any model running locally |

## Provider Selection

```python
@llm_test(
    expect.is_not_empty(),
    provider="openai",
    model="gpt-4o-mini",
)
def test_openai(llm): ...
```

## Default Provider

<Callout>
  If no provider is specified, assertllm checks for available API keys in this order:
</Callout>

1. `OPENAI_API_KEY` → OpenAI
2. `ANTHROPIC_API_KEY` → Anthropic
3. Ollama (localhost) → Ollama
