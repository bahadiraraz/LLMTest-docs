import { Callout } from 'nextra/components'

# Text Assertions

<Callout type="info">
  All text assertions are **deterministic** — no LLM calls, instant results.
</Callout>

## contains

Checks that the output contains a substring.

```python
expect.contains("Paris")
expect.contains("paris", case_sensitive=False)
```

## not\_contains

Checks that the output does **not** contain a substring.

```python
expect.not_contains("London")
```

## matches\_regex

Checks that the output matches a regular expression.

```python
expect.matches_regex(r"\d{4}")  # contains a 4-digit number
```

## valid\_json

Checks that the output is valid JSON. Optionally validates against a JSON schema.

```python
expect.valid_json()
expect.valid_json(schema={
    "type": "object",
    "properties": {
        "name": {"type": "string"},
        "age": {"type": "integer"}
    },
    "required": ["name", "age"]
})
```

## length\_between

Checks that the output word count is within a range.

```python
expect.length_between(10, 100)
```

## starts\_with

```python
expect.starts_with("The")
```

## ends\_with

```python
expect.ends_with(".")
```

## is\_not\_empty

```python
expect.is_not_empty()
```

## one\_of

Checks that the output exactly matches one of the given options.

```python
expect.one_of(["yes", "no", "maybe"])
```

## similar\_to

Word overlap similarity (Jaccard). No LLM call.

```python
expect.similar_to("Paris is the capital of France", threshold=0.7)
```

## contains\_all

Must contain **all** substrings.

```python
expect.contains_all(["Paris", "France", "capital"])
```

## contains\_any

Must contain **at least one** substring.

```python
expect.contains_any(["Paris", "Lyon", "Marseille"])
```

## structured\_output

Validates output as a Pydantic model.

```python
from pydantic import BaseModel

class CityInfo(BaseModel):
    name: str
    country: str
    population: int

expect.structured_output(CityInfo)
```
